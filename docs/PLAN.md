# 4주 팀 경쟁 챌린지 v0 — 전체 운영 계획서

## 1. 시스템 핵심 구조

| 항목 | 설명 |
|------|------|
| 팀 구성 | 2인 1팀 |
| 챌린지 기간 | 4주 (시즌) |
| 달성률 | 개인 목표 기반 누적 계산 (100% cap) |
| 주간 순위 | 매주 리셋, 최종 순위는 4주 합산 |
| 주 초 | 팀 미션 입력 |
| 주 말 | Admin 수동 마감 |
| 하위팀 | 하위 2팀에 벌칙 미션 부여 |
| 누락 처리 | 입력 누락 시 직전 주 데이터 유지 |

## 2. 인증 구조

| 역할 | 인증 방식 |
|------|----------|
| 일반 유저 | **카카오 로그인** (OAuth2) |
| Admin | **이메일 + 비밀번호** (별도 계정) |

## 3. USER (팀원) 전체 플로우

### Phase 0 — 최초 참여
1. 초대 링크 접속
2. 카카오 로그인으로 계정 생성
3. 시작 인바디 입력
4. 목표 설정 (1~2개 선택)
5. 팀 배정 확인

> 여기까지 완료해야 Week 1 시작 가능

### Phase 1 — 매주 월요일 (주 초)
- 팀 탭 접속
- "이번 주 팀 미션 입력" 화면 노출
- 팀원 중 1명이 입력, 다른 1명은 확인
- 입력 내용:
  - 미션 유형 선택 (템플릿 기반)
  - 목표 수치 입력
  - 예: 운동 횟수 5회 / 러닝 거리 20km / 식단 인증 12회
- **미션은 팀 단위 1개**

### Phase 2 — 주간 진행 중

**개인 영역:**
- 인바디 입력 (주 1회 이상)
- 개인 달성률 실시간 확인

**팀 영역:**
- 팀 평균 점수 실시간 표시
- 팀 미션 진행률 표시
- 팀원별 미션 인증 업로드

### Phase 3 — 주간 마감 전
- 마감 카운트다운 표시
- 미입력자 알림

### Phase 4 — Admin 마감 후
- 이번 주 팀 순위 확인
- 팀 점수 확인
- 하위 2팀 여부 확인
- 팀 미션 성공/실패 확인

### Phase 5 — 하위팀일 경우
- 룰렛 미션 확인
- 팀원 2명 모두 수행
- 인증 업로드
- Admin 승인 대기

## 4. ADMIN 전체 플로우

### Phase 0 — 챌린지 생성
- 챌린지 생성 (이름, 설명)
- 기간 설정 (4주)
- 팀 구성
- 목표 설정 마감일 설정

### Phase 1 — 주 초 관리
- 팀 미션 입력 여부 확인
- 미입력 팀 알림
- 미션 승인 (필요 시)

### Phase 2 — 주간 진행 중
- 전체 팀 점수 모니터링
- 미입력 개인 확인

### Phase 3 — 주간 마감 (핵심)
Admin이 `[Close Week N]` 버튼 클릭 시 시스템 자동 실행:
1. 개인 달성률 계산
2. 입력 누락자 → 직전 주 유지 적용
3. 팀 점수 계산
4. 팀 순위 산정
5. 하위 2팀 확정
6. **결과 스냅샷 저장 (수정 불가)**

### Phase 4 — 하위팀 미션 룰렛
- 하위 2팀 자동 표시
- "룰렛 돌리기" 버튼
- 팀별 벌칙 미션 배정
- 확정 저장

### Phase 5 — 미션 인증 관리
- 팀원 2명 모두 인증 확인
- 완료 체크
- 미완료 시 경고 표시

### Phase 6 — 시즌 종료
- FinalScore 계산 (4주 합산)
- 최종 순위 산정
- 우승팀 확정

## 5. 주간 타임라인

| 시점 | 활동 |
|------|------|
| 월~화 | 팀 미션 입력 |
| 수~일 | 개인 인바디 입력 + 팀 미션 수행 |
| 일요일 밤 | Admin 마감 실행 |
| 마감 직후 | 순위 공개 → 하위 2팀 룰렛 → 미션 배정 |

## 6. 데이터 흐름

```
[입력] User → DB 저장
[계산] Admin 마감 → 서버 계산 → Snapshot 저장
[조회] User는 Snapshot만 조회
```

> **주간 마감 후 값은 절대 수정 불가**

## 7. 화면 구조

### USER 탭
| 탭 | 설명 |
|----|------|
| 홈 | 대시보드, 주간 현황 |
| 팀 | 팀 미션, 팀원 상태 |
| 인증 | 미션 인증 업로드 |
| 내정보 | 개인 목표, 인바디, 프로필 |

### ADMIN 탭
| 탭 | 설명 |
|----|------|
| 대시보드 | 전체 현황 |
| 주간 마감 | Close Week 실행 |
| 순위표 | 주간/최종 순위 |
| 미션 관리 | 룰렛, 벌칙 미션 |
| 사용자 관리 | 유저/팀 관리 |

## 8. 역할별 책임

| 역할 | 책임 |
|------|------|
| 개인 | 목표 관리 + 인바디 입력 |
| 팀 | 주간 미션 설정 + 수행 |
| Admin | 마감 + 순위 확정 + 미션 부여 |
