# 화면 구조 및 라우팅

## USER 화면

| 경로 | 화면명 | 설명 |
|------|--------|------|
| `/` | 홈 | 주간 현황, 팀 점수, 마감 카운트다운 |
| `/login` | 로그인 | 카카오 로그인 버튼 |
| `/onboarding` | 온보딩 | 최초 참여: 인바디 입력 + 목표 설정 |
| `/team` | 팀 | 팀 미션 입력/현황, 팀원 상태 |
| `/verify` | 인증 | 미션 인증 업로드, 인증 내역 |
| `/profile` | 내정보 | 개인 목표, 인바디 기록, 프로필 |
| `/result` | 주간 결과 | 마감 후 순위, 팀 점수 |

## ADMIN 화면

| 경로 | 화면명 | 설명 |
|------|--------|------|
| `/admin/login` | Admin 로그인 | 이메일 + 비밀번호 |
| `/admin` | 대시보드 | 전체 현황, 주간 통계 |
| `/admin/weekly-close` | 주간 마감 | Close Week 버튼, 마감 실행 |
| `/admin/rankings` | 순위표 | 주간/최종 순위 |
| `/admin/missions` | 미션 관리 | 룰렛, 벌칙 미션 배정/승인 |
| `/admin/users` | 사용자 관리 | 유저/팀 관리 |
| `/admin/challenges` | 챌린지 관리 | 챌린지 생성/설정 |
